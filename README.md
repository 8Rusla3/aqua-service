# Aqua Service Inc Website

Website for Aqua Service Inc—Toronto's trusted contractor fixing wet basements, backed-up drains, foundation leaks, and home renovations. We diagnose honestly, quote fairly, and deliver guaranteed work.

## Structure

- `index.html` - Main HTML file with problem-focused, contractor language
- `css/style.css` - Stylesheet
- `js/script.js` - JavaScript for mobile menu
- `assets/` - Folder for images and other assets
- `.htaccess` - Security and server configuration
- `robots.txt` - SEO configuration

## Features

- Fully responsive design optimized for mobile devices
- Mobile-first approach with breakpoints at 640px, 768px, and 1024px
- Sticky navigation with mobile hamburger menu
- Contact form ready for Netlify deployment
- Clean, professional design with CSS custom properties

## Deployment

1. Upload all files to your web server
2. For Netlify: Enable form handling in site settings
3. Update contact information and phone numbers
4. Add real images to the assets folder
5. Update robots.txt with your actual domain

## Security

- Security headers configured in .htaccess
- XSS protection, content type sniffing prevention
- Frame options set to deny
- Directory listing disabled

## Customization

- Colors and variables defined in CSS :root
- Easy to modify fonts, spacing, and layout
- Add images by placing them in assets/ and updating HTML
